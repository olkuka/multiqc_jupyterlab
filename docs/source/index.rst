
multiqc_jupyterlab
=====================================

Version: |release|

JupyterLab extension for MultiQC


Quickstart
----------

To get started with multiqc_jupyterlab, install with pip::

    pip install multiqc_jupyterlab

or with conda::

    conda install multiqc_jupyterlab
